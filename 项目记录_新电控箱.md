- **前端**：在浏览器中操作 DOM、处理事件，结合 React、Vue 等框架构建复杂的单页应用（SPA）。
- **后端**：通过 **Node.js** 在服务器上运行，处理高并发网络请求，开发 API 服务。
- **移动端**：使用 React Native、Flutter 的部分逻辑、或原生内嵌 JS 引擎开发 App。
- **桌面端**：用 Electron 等框架开发跨平台桌面应用（如 VS Code、Slack）。
- **甚至物联网和嵌入式设备**也能运行 JavaScript。

它的核心特点可以概括为：**单线程、异步非阻塞、事件驱动、原型继承、动态类型**。

第一阶段：打好坚实的地基 (1 – 2 个月)

- **核心语法**：变量（`let`、`const`，忘掉 `var`）、数据类型、运算符、条件判断、循环、函数（声明式、箭头函数）。
- **数据结构**：数组及其方法（`map`、`filter`、`reduce`是重中之重）、对象。
- **JavaScript 怪癖**：理解 `this` 指向、作用域链、闭包、原型链。这是面试重点，也是真正理解的标志。
- **动手**：在浏览器控制台直接写代码，尝试解决一些小问题，比如写一个计算器、做一个简单的问答列表。

第二阶段：拥抱现代 JavaScript 与异步 (1 个月)

- **ES6+ 核心特性**：熟练掌握解构、模板字符串、展开运算符、`Set/Map`、`Class` 基本用法、模块化。
- **异步编程**：这是 JS 的灵魂！从回调函数概念起步，重点攻克 `Promise` 和 `async/await`。用它去请求远程 API 数据。
- **项目**：写一个天气查询页，输入城市，调用免费天气 API 显示结果。你会深刻理解异步。

第三阶段：与浏览器交互 & 深入一个框架 (2 – 3 个月)

- **DOM 与 BOM**：学会用原生 `document.querySelector` 等方法操作页面元素、处理表单、监听事件。
- **做个完整的原生项目**：一个待办事项应用（Todo App），能增删改查，数据存到 `localStorage`。做完你会很有成就感。
- **选择你的第一个框架**：推荐从 **Vue.js** 或 **React** 入手。Vue 渐进式易上手，React 生态更庞大。学习它们的核心思想：组件化、状态管理、数据驱动视图。用你选的框架重写之前的 Todo App。

第四阶段：迈向全栈与工程化 (持续)

- **学习 Node.js 基础**：尝试用 Express 或 Koa 搭建一个简单的 API 服务器，连上数据库（如 MongoDB 或 MySQL）。你会发现前端造出的数据请求，居然能被自己写的后端响应，那一瞬间就全通了。
- **了解工程化工具**：知道 `npm` 怎么管理包，`Vite` 或 `Webpack` 是怎么打包项目的，Git 如何进行版本控制。
- **构建完整项目**：比如一个博客系统、一个在线书店。从设计前端页面，到搭建后端接口，再到部署上线。

Node.js 内部采用**事件驱动、异步非阻塞**模型。你可以把程序想象成一张由**节点**（事件、回调函数、任务）组成的**图**。

- 一个任务完成后，会**连接**到下一个回调函数（下一个节点）
- 这些节点之间通过事件循环连通

因此，Node.js 中的程序就是这些异步任务节点互相连接构成的图。



把每个 Node.js 应用看作网络中的一个**节点**。
